WHAT THIS SYSTEM DOES
================================================================================

This is a complete, production-ready, real-time risk management system that:

1. Continuously generates realistic hospital accounting transactions
2. Streams data through Apache Kafka for reliable processing
3. Analyzes each transaction using multi-dimensional risk scoring
4. Detects anomalies and suspicious patterns in real-time
5. Generates immediate alerts for high-risk transactions
6. Stores all data in PostgreSQL for historical analysis
7. Visualizes everything in a real-time web dashboard
8. Scales horizontally and vertically for any load
9. Runs in Docker containers for easy deployment
10. Works out-of-the-box with a single command
